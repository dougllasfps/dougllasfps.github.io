# Ferramentas Online

Portal de calculadoras e conversores desenvolvido em HTML, CSS e JavaScript puro.

## Como executar

1. Abra o arquivo `index.html` no navegador.
2. Para melhor navegação entre páginas, opcionalmente sirva a pasta com um servidor local simples (ex.: extensão Live Server).

## Estrutura

- `index.html`
- `pages/`
- `css/`
- `js/`
- `assets/`
- `data/`
